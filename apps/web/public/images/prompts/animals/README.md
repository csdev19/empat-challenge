# Animal Prompt Images

This directory contains placeholder images for the word-picture choice game animal prompts.

## Required Images

The following images are needed (300x300px, PNG format):

- `dog.png` - Picture of a dog
- `cat.png` - Picture of a cat
- `bird.png` - Picture of a bird
- `fish.png` - Picture of a fish
- `cow.png` - Picture of a cow
- `horse.png` - Picture of a horse
- `pig.png` - Picture of a pig
- `duck.png` - Picture of a duck

## Image Requirements

- **Size**: 300x300px square
- **Format**: PNG
- **Style**: Clear, simple, high contrast
- **Consistency**: Similar visual style across all images

## Temporary Placeholders

For development, you can use:

1. Colored rectangles with text labels (e.g., a blue square with "DOG" text)
2. Free images from Unsplash or Pexels
3. AI-generated images
4. Custom illustrations
